WCMC GeoNode Help

This page provides helpful information about how to use the WCMC GeoNode. You can use the sidebar links to navigate to what you want to know.

WCMC Geonode is our official dataset repository and it provides access both for internal and external users. It also provides a map editing application that allows users to browse existing maps and contribute their own. It is also possible to access it through WFS and WMS

Layers

All GIS information is stored under the Layers label.

Browsing and Searching

All layers can be browsed and searched by clicking on the Layers tab. You will be able to filter results by Text, Type, Categories, Keywords, Owners, Date, Regions and Extent. All data can be downloaded in a variety of formats, for use in other applications.

Uploading layers

The instructions and SOPs to upload layers can be found on this page.

Maps

WCMC Geonode allows users to create and share maps with one another. Help on Browsing, Creating, Exporting and Remixing Maps can be found on the Maps Help Page
