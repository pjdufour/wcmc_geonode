WCMC GeoNode Help

This page provides helpful information about how to use the WCMC GeoNode. You can use the sidebar links to navigate to what you want to know.

WCMC Geonode is our official dataset repository and it provides access both for internal and external users. It also provides a map editing application that allows users to browse existing maps and contribute their own. It is also possible to access it through WFS and WMS


Geonode is the central point for UNEP-WCMC's spatial data. This data is available on the Layers Section. You can find information about how to access or upload it on the Layers' Help Page.


On the Maps Section you can create and share a visualization with the information you can access. On the Maps' Help Page you can find information on how to build those maps.


For other data there is the Documents Section. You can add different formats and also link them to Spatial Data. More information on the Documents' Help Page.

For information on users roles within Geonode you can check People's Help Page.


You can create groups or browse through your Programme data and people on the Groups Section. Information can be found on Groups Page.


On Geonode you should use UNEP-WCMC Naming Standards
