People

In the people tab you can browse through all the users and check their layers, maps and documents.

There are three different types of user: Guest, Regular User, Data Champion and Data Manager. Each of them have different levels of access to UNEP-WCMC Data Portal.

Guest User

A guest user can be anyone that has browsed to UNEP-WCMC Data Portal. This person cannot add datasets but he/she can have access to open datasets and read the owner's profile.

Regular User

A regular user is a member of UNEP-WCMC that has not Programme admin privileges. Apart from that this user has access to almost all UNEP-WCMC Geoonode features including uploading data, setting permissions, accessing data and creating and manage project groups.

Data Champions and Heads of Programme

The Data Champions and Heads of Programme can access the same features as a regular user but they can also manage their Programme's group, including adding and removing users from it and giving managing permissions to other users.

Data Manager

The data manager has a full admin access and can manage all datasets, groups and users.
