Groups

In order to make easier to define who has access to a dataset UNEP-WCMC has the groups feature. We have base groups that match the Centre's Programmes but they can also be created on demand for projects or inter-programme groups of users.

Creating and Editing a Group

Groups can be created after cliccking in a button on its page. To edit you will need to browse and click on Edit Programme Details You will need to set a title (it may not start with a number), choose a logo and write a description. It is also possible to set an email to contact the group manager and to set keywords. The most important setting is setting the access type, that can be:

  • Public - All users can view and join the group
  • Public (invite only) - All users can view the group but joining is only by invitation
  • Private - Only group members can view and joining is only by invitation

Managing Group Mmebers

Users can have two different roles within a group:

  • Members: Users that can access group's datasets and documents
  • Managers: Users that can access group's datasets, maps and documents, manage members edit and delete the group

The manager can add members from all UNEP-WCMC Data Portal users and set their roles.

Viewing activities

It is possible to browse through all the changes made within a group clicking on the Activity Feed link.
